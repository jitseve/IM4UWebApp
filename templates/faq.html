Frequently Asked Questions

What is IdrottsMätta4U?

IM4U is a mobile websystem that synchronizes with inertial measurement units (IMU) sensors to acquire acceleration data. Using this raw data IM4U will provide your step frequency and ground contact times.
You are also upload data acquired with the MoveSense mobile application, make sure that you save the data in one file.

With what events can I use IdrottsMätta4U?

The current IM4U version only works for running and sprint data. The IM4U development team is working on developing algorithms that will cover high jump, long jump, pole vault and much more.

How can I benefit from using IdrottsMätta4U?

IM4U provides accurate information about step frequency and ground contact times. Knowing that fro springting; shorter ground contact times result in higher speeds and thus faster sprinting.
Measuring ground contact times will allow users to track performance improvement while having important insight into where they can improve.

Can I use the IdrottsMätta4U web app with any IMU sensor?

The IM4U websystem has been calibrated using the sensors made by MoveSense.
However, any inertial measurement unit that can output IMU6 data in a csv format should work with our algorithms.

Can IdrottsMätta4U be used by multiple users with different accounts?

At the moment we don't have accounts yet.

Do I get a cloud space for storing my data history?

At the moment we don't allow for storage yet

Can I extract my saved data from IdrottsMätta4U?

Currently the output information is only accesible through the IM4U websystem

Is the IdrottsMätta4U websystem compatible with any webbrowser?

Everything except for the recording of data is compatible with any webbrowser.
Recording data requires a bluetooth connection of the webbrowser, which is currently only supported by chromium webbrowsers (chrome etc.).

Can I change the language of the websystem?

No, the current version only supports English.

How can I start with IdrottsMätta4U?

Buy a Movesense and get going!
